## Overview

Senamby is a desktop application for operating plants through reusable plugins. A plant combines:

- sensors
- actuators
- one driver plugin
- zero or more controller plugins

The application lets you define the plant model, connect it to a runtime, plot values in real time, and import/export JSON-based artifacts.

## Who This Is For

- operators configuring plants and running experiments
- integrators building driver plugins for devices and protocols
- control engineers creating controller plugins and bindings

## Main Workflows

1. Load or create the plugins your plant needs
2. Create a plant or import a plant JSON file
3. Configure variables, driver settings, and controller bindings
4. Connect the plant to start the runtime
5. Adjust setpoints and controller configuration while monitoring the plots
